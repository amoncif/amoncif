## Ce que je fais
Je construis et je maintiens des applis web **orientées business** :
- **e-commerce** (tunnel, perf, fiabilité),
- **CRM / apps métiers** (workflows, rôles/droits, exports, recherche),
- **RH** (process, validations, reporting),
- **sites à fort trafic** (latence, stabilité, requêtes, cache applicatif),
- et en ce moment : **outillage & intégrations autour de CyberArk / PSM** (accès privilégiés, gouvernance, audit).

## Ma manière de bosser
- je préfère les **contrats clairs** (API, DTO/serializer groups, erreurs, pagination)
- je traite la perf comme une feature : **plans d’exécution**, index, pagination, réduction N+1
- je vise du code qui se relit : nommage, découpage, responsabilité claire, tests utiles
- je sais reprendre un existant sans tout casser : refacto progressif, “safe changes”, observabilité
